# AI Career Mentor

An AI-powered career guidance platform that helps students discover suitable career paths using Machine Learning.

## Features

- Career prediction using ML
- Skill gap analysis
- Learning roadmap suggestions
- Internship recommendations
- Salary insights
- Resume improvement tips

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Structure

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Model Training
- Career Prediction Engine
- Visualization Dashboard

## Dataset

The project uses a realistic career dataset containing skills, interests, certifications, projects, internships, and career information.
